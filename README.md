# 🏍️ Aero Apex Motorcycle Accessories

Online store for premium motorcycle accessories. Built with Flask + SQLite. Includes a catalog of over 370 products, a full-featured admin panel, and WhatsApp ordering.

## 🚀 Technologies
- Python + Flask
- SQLite + SQLAlchemy
- Vanilla HTML/CSS/JS
- Gunicorn (production)
- Railway (hosting)

## ✨ Features
- Catalog with over 370 products
- Full admin panel
- WhatsApp ordering
- Dark racing-style design
- Categories: Mirrors, Intercoms, Lighting, Grips, Sliders, Apparel, and more

## ⚙️ Run locally
```bash
pip install -r requirements.txt
python app.py
```

- Store: http://localhost:5000
- Admin: http://localhost:5000/admin
