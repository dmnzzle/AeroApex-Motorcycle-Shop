# 🏍️ Aero Apex Motorcycle Accessories

## Ejecutar localmente
```bash
pip install -r requirements.txt
python app.py
```
- Tienda: http://localhost:5000
- Admin:  http://localhost:5000/admin
- Usuario: admin / aeroapex2024

## Desplegar en Railway (gratis, en la nube)

1. Crea cuenta gratis en **railway.app**
2. Haz clic en **"New Project"** → **"Deploy from GitHub repo"**
   - O usa: **"Deploy from local"** y sube esta carpeta
3. Railway detecta Python automáticamente
4. La tienda queda en internet en ~2 minutos
5. URL pública gratis incluida

## Panel Admin
- `/admin` → Dashboard
- `/admin/productos` → Gestionar productos
- `/admin/categorias` → Gestionar categorías  
- `/admin/media` → Subir videos y GIFs
- `/admin/diseno` → Cambiar colores y estilos
- `/admin/pagina-inicio` → Editar textos del hero
- `/admin/config` → WhatsApp, redes sociales
- `/admin/secciones` → Imágenes de secciones
